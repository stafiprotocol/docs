# Staking SOL with StaFi
import { Callout } from 'nextra/components'

<Callout type="error">
StaFi has taken the rSOL service offline. No further rewards will be distributed.

Stakers should withdraw their assets using the `rsol-relay` CLI tool by following the steps below.

**Case 1: You have an existing unstake account (unbonding in progress)**

Wait for the unbonding period to end, then configure `config_staker_example.toml`:

```toml
UnstakeAccount   = "<your_unstake_account_pubkey>"  # your unstake account
RecipientAccount = "<your_recipient_pubkey>"         # address to receive SOL
```

Run:

```bash
rsol-relay staker withdraw --config <config_path>
```

**Case 2: You still hold rSOL in your wallet**

Configure `config_staker_example.toml`:

```toml
EndpointList = ["https://api.mainnet-beta.solana.com"]
KeystorePath = "./keys/solana_keys.json"

StakeManagerProgramID = "EiweRuKbjvwkRhraQ5vreVd3xon3JYfB7efiPDhFuVeL"
StakeManagerAddress   = "CxwWmrREkaugoJ9s2uCX4TaUuWERPo2yLbP7TrD3qo9k"

FeePayerAccount    = "<fee_payer_pubkey>"
BurnRsolFrom       = "<rSOL_token_account>"
BurnRsolAuthority  = "<authority_pubkey>"
RecipientAccount   = "<recipient_pubkey>"
Amount             = <rSOL_amount_lamports>
```

Run:

```bash
rsol-relay staker unstake --config <config_path>
```

Your SOL will be returned to your wallet immediately.
</Callout>
